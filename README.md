# twentySix — aplikacja mobilna

Aplikacja mobilna **twentySix** (React Native / Expo): sędziowanie turniejów, quick game, znajomi. Współpracuje z API Laravel (`twentysix-backend`).

## Dokumentacja

- **Wizja produktu:** `../twentysix-backend/docs/product.md`
- **Logika biznesowa:** `../twentysix-backend/LOGIKA_BIZNESOWA.md`
- **Stan implementacji vs MVP:** [`IMPLEMENTED_FEATURES.md`](IMPLEMENTED_FEATURES.md)

## Uruchomienie

```bash
npm install
npx expo start
```

Skonfiguruj URL API w aplikacji zgodnie z backendem twentySix.
